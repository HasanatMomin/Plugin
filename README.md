# McPvP

Competitive PvP plugin for Paper 26.1.2.

## Features
- queueing
- matchmaking
- duel sessions
- persistent profiles and rankings
- leaderboard
- SQLite storage

## Commands
- `/mcpvp`
- `/mcpvp leaderboard`
- `/queue join`
- `/queue leave`
- `/queue status`
- `/duel <player>`

## Build
Requires Java 21.

./gradlew build
